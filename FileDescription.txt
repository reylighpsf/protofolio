📂 Struktur Direktori:
├── dist/            ⭐ Ini adalah build produksi (direkomendasikan untuk penggunaan langsung)
│   ├── index.html   → Buka file ini di browser untuk melihat pratinjau situs web
│   ├── assets/      → File CSS, JS, dan sumber daya lainnya yang sudah dibundel
│   └── ...
├── src/             📝 Direktori kode sumber (untuk pengembangan dan modifikasi)
│   ├── components/  → Komponen React
│   ├── App.jsx      → Komponen aplikasi utama
│   ├── index.js     → File entri
│   └── ...
├── package.json     📦 Konfigurasi ketergantungan proyek
├── README.md        📖 Dokumentasi proyek
└── ...

🚀 Memulai Cepat (Dua Opsi):
Opsi 1: Penggunaan Langsung (Direkomendasikan untuk Pemula)
1. Arahkan ke direktori dist/
2. Klik dua kali file index.html
3. Situs web akan terbuka di browser Anda!

Opsi 2: Mode Pengembangan (Untuk Pengguna dengan Pengalaman Koding)
1. Pastikan Node.js terinstal (versi yang direkomendasikan: 16+)
2. Buka terminal/command line di direktori root proyek
3. Jalankan: npm install
4. Jalankan: npm start
5. Server pengembangan akan terbuka secara otomatis di browser Anda

💡 Tips:
- Direktori dist/ berisi build yang dioptimalkan dan dapat langsung diterapkan ke server
- index.html di tingkat root bukan versi final — selalu gunakan yang ada di dalam dist/
- Untuk memodifikasi kode, edit file di dalam src/ dan bangun ulang proyek
- Proyek ini menggunakan tumpukan teknologi TypeScript dan Tailwind CSS
